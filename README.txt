Requires at least: 4.3.1


== Description ==

Add a Panoramic Video with Krpano(1.81) to Wordpress posts or pages by embedding it.


== Installation ==
1. Unzip archive to the \'/wp-content/plugins/\' directory.

2. upload panoramic video and thumbnails to the \'/wp-content/plugins/krpano-video-plugin/krpano/video\' directory. Videos play on ios recommand 1024x512, and others 1920x960.

2. Activate the plugin through the \'Plugins\' menu in WordPress.
